# MST Blockchain

Secure, scalable, and sustainable Web3 infrastructure built in Pune, Maharashtra.

## Overview

MST Blockchain is India's first Layer 1 blockchain platform designed for high-performance decentralized applications.

### Key Features

- **Scalable**: High throughput with sub-second finality.
- **Secure**: Advanced cryptographic security protocols.
- **Eco-friendly**: Sustainable consensus mechanism.

## Development

### Prerequisites

- Node.js (v18+)
- npm or bun

### Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
